      Mettre à jour le nombre de feuilles total

      UpdateTotalSheetCountMise en page

      Met à jour le nombre de feuilles total dans les jeux de feuilles sélectionnés. Le paramètre SS_Nombre de feuilles total doit être présent dans le projet. Seules les feuilles dont la propriété Figure dans la liste des feuilles est cochée seront comptabilisées.

      Update Total Sheet Count

      UpdateTotalSheetCountSheet

      Updates the total sheet count in the selected sheet sets. The SS_Nombre de feuilles total parameter must be present in the project. Only sheets whose Appears in Sheet List property is checked will be counted.

      Repositionner automatiquement le texte des dimensions dans le cadrage

      DimensionTextMoveInsideCropRegionAnnotations

      Repositionne le texte des dimensions sélectionnées à l'intérieur du cadrage de la vue (le plus proche de la position initiale). Cette commande permet d'éviter que le texte soit mal représenté en PDF (survient lorsque le texte est entre 2 zones). Fonctionne seulement avec les cotes dont les segments sont des lignes droites.

      Automatically Reposition Dimension Text Inside Crop Region

      DimensionTextMoveInsideCropRegionAnnotations

      Repositions the text of selected dimensions inside the view crop region, as close as possible to its original position. This command helps prevent text from being incorrectly represented in PDFs, which can occur when the text is between two regions. Only works with dimensions whose segments are straight lines.

      Repositionner le texte d'une dimension

      DimensionTextMoveToPointAnnotations

      Permet de repositionner manuellement le texte d'une dimension sélectionnée. Utile pour les dimensions dont le texte est positionné entre 2 zones de cadrage. Dans cette situation, le texte d'une cote ne peut pas être déplacé avec la poignée standard.

      Reposition Dimension Text

      DimensionTextMoveToPointAnnotations

      Allows the text of a selected dimension to be manually repositioned. Useful for dimensions whose text is positioned between two crop regions. In this situation, the dimension text cannot be moved with the standard grip.
